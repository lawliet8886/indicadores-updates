# Indicadores CAPS Updates

Canal oficial de atualizações do aplicativo **Indicadores CAPS**.

Este repositório existe apenas para distribuir **releases públicas** do programa.
O código-fonte principal **não é publicado aqui**.

## Como as atualizações funcionam

As novas versões do aplicativo são distribuídas na aba **Releases** deste repositório.
O instalador mais recente será anexado a cada release publicada.

## Arquivos esperados em cada release

- `IndicadoresCAPS_Setup.exe` — instalador principal para Windows
- `CHANGELOG.json` — changelog da versão (opcional, mas recomendado)
- `SHA256.txt` — hash do instalador (opcional)

## Observações

- Este repositório deve permanecer leve.
- Não enviar binários diretamente para a raiz do repositório.
- Os arquivos do programa devem ser anexados em **Releases**, não versionados no Git.
